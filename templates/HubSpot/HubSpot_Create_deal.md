## Model

### Create deal

### Using send method


```python
send_deal = {"properties": 
                  {
                    "dealstage": dealstage,
                    "dealname": dealname,
                    "amount": amount,
                    "closedate": closedate,
                    "hubspot_owner_id": hubspot_owner_id,
                   }
                 }

deal1 = hubspot.connect(HS_API_KEY).deals.send(send_deal)
```

### Using create method


```python
deal2 = hubspot.connect(HS_API_KEY).deals.create(
    dealname,
    dealstage,
    closedate
)
```

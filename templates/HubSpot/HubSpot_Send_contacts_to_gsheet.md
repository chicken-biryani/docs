## Model

### Get all contacts


```python
df_contacts = hubspot.connect(HS_API_KEY).contacts.get_all(properties_list)
```

## Output

### Send contacts to gsheet


```python
gsheet.connect(spreadsheet_id).send(
    sheet_name=sheet_name,
    data=df_contacts,
    append=False
)
```

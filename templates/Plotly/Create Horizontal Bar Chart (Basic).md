## Input

### Import library


```python
import plotly.graph_objects as go
```

## Model

### Create the plot


```python
fig = go.Figure(go.Bar(
            x=[20, 14, 23],
            y=['giraffes', 'orangutans', 'monkeys'],
            orientation='h'))
```

## Output

### Display result


```python
fig.show()
```
